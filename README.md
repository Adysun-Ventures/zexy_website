# 🌍 Zexy Marketing & Onboarding Website

## 📌 Project Overview
The `zexy_website` serves fundamentally as the top-of-funnel conversion hub targeting digital brand awareness. It acts as the explicit landing portal onboarding Creators looking to scale operations, and Fans looking to discover platform utilities organically.

**Target User**: Public / Unregistered Users

## 🛠 Tech Stack
- **Framework**: Next.js (Static Site Generation / ISR capabilities) or Astro (for max speed)
- **Animations**: Framer Motion / GSAP
- **Styling**: Tailwind CSS

## ✨ Key Features
- **Conversion Pipelines**: Distinct pathways organically wrapping "Become a Creator" tracking analytics directly.
- **SEO Optimization**: High-performance semantic architecture defining rapid Time-to-Interactive bounds flawlessly.
- **Showcase Engine**: Smooth staggered animations outlining the exact UI layouts of the Mobile and Consumer applications natively projecting trust securely.

## 🏗 Architecture Role
- Operates totally isolated. Only utilizes highly cached `/public` API endpoints tracking top-creator leaderboards optionally driving high conversion metrics.

## 📂 Basic Folder Structure
```text
zexy_website/
├── src/
│   ├── pages/            # SEO indexed routes (/creators, /pricing, /about)
│   ├── components/       # Heavy animation-first landing sections natively
│   └── styles/           # Core brand identity bounds tracking strictly.
└── public/               # High-res marketing media outputs.
```

## 🚀 Setup Instructions
1. **Dependencies**: `npm install`
2. **Boot**: `npm run dev`

## 🔌 API & Integration Notes
- Entirely unauthenticated read-only pipelines optionally projecting top trending Creator usernames organically tracking platform health effectively.
